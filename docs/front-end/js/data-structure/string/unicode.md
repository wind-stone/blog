# Unicode

## 常用 Unicode 字符

[Unicode字符平面映射](https://zh.wikipedia.org/wiki/Unicode%E5%AD%97%E7%AC%A6%E5%B9%B3%E9%9D%A2%E6%98%A0%E5%B0%84)

- `\u2070`~`\u209F`：上标和下标
- `\u2190`~`\u21FF`：箭头
- `\u2200`~`\u22FF`：数学运算子
- `\u2460`~`\u24FF`：带圈字母和数字
- `\u2500`~`\u257F`：制表符
- `\u25A0`~`\u25FF`：几何图形
- `\u25A0`~`\u25FF`：几何图形
