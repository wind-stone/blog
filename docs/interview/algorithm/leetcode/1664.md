# 1664. 生成平衡数组的方案数[中等]

- 题目：[1664. 生成平衡数组的方案数](https://leetcode.cn/problems/ways-to-make-a-fair-array/description/)
- 题解：[生成平衡数组的方案数](https://leetcode.cn/problems/ways-to-make-a-fair-array/solutions/2078340/sheng-cheng-ping-heng-shu-zu-de-fang-an-0mkaj/)

## AC 代码

前提：

删除某个下标为 i 的元素后，其后元素的下标都要 -1（即奇数下标变成偶数下标，偶数下标变成奇数下标），想要达成平衡数组，需要在删除下标 i 的元素后，其之前奇数下标元素的总和 + 其后偶数下标元素的总和 = 其之前偶数下标元素的总和 + 其后奇数下标元素的总和。

```js
var waysToMakeFair = function (nums) {
    let odd1 = 0; // 记录当前下标 i 左侧（不包含 i）所有奇数下标元素的累加和。
    let even1 = 0; // 记录当前下标 i 左侧（不包含 i）所有偶数下标元素的累加和。
    let odd2 = 0; // 记录当前下标 i 右侧（包含 i 及之后）所有奇数下标元素的累加和。
    let even2 = 0; // 记录当前下标 i 右侧（包含 i 及之后）所有偶数下标元素的累加和。
    for (let i = 0; i < nums.length; ++i) {
        if ((i & 1) !== 0) {
            // 计算初始奇数下标的和（整个数组奇数下标的和）
            odd2 += nums[i];
        } else {
            // 计算初始偶数下标的和（整个数组偶数下标的和）
            even2 += nums[i];
        }
    }
    let res = 0;
    for (let i = 0; i < nums.length; ++i) {
        if ((i & 1) != 0) {
            // 如果当前是奇数下标，则其右侧奇数和为：
            odd2 -= nums[i];
        } else {
            // 如果是偶数，则其右侧偶数下标的和为：
            even2 -= nums[i];
        }

        // 假设当前删除下标 i 对应的元素
        if (odd1 + even2 === odd2 + even1) {
            // 关键点：如果删除当前的元素后能达成平衡数组，则会满足上述条件
            ++res;
        }

        // 判断完之后，要把当前下标的元素值加到 之前的 odd1 或 even1 上
        if ((i & 1) !== 0) {
            odd1 += nums[i];
        } else {
            even1 += nums[i];
        }
    }
    return res;
};
```
