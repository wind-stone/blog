# 【初级】函数参数值传递问题

题 1： 代码执行完后`a`、`b`的值是什么？

```js
let a = [1, 2, 3]
let b = 1

const foo = (obj, num) => {
    obj = [5, 4, 3]
    obj[1] = 999
    obj = [5, 4, 3]
    num = 2
}

foo(a, b)
console.log(a, b);

// 结果： [1, 2, 3]  1
```

题2：给出`a(1)`返回值

```js
function a(x, f=()=>x) {
    var x;
    y = x;
    x = 2;
    return [x,y,f()];
}
a(1)
```
