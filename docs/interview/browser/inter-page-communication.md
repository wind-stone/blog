# 浏览器页面间通信

| 方法               | 是否需要窗口引用 | 是否支持跨域 | 适用场景                               |
| ------------------ | ---------------- | ------------ | -------------------------------------- |
| BroadcastChannel   | 否               | 否           | 现代浏览器下的同源广播通信             |
| localStorage       | 否               | 否           | 对兼容性要求高的简单通信场景           |
| window.postMessage | 是               | 是           | 有明确父子关系的窗口间通信，或需要跨域 |
| SharedWorker       | 否               | 否           | 复杂通信场景，需要共享状态或计算       |
| Service Worker     | 否               | 否           | 需要离线支持或后台同步的复杂场景       |
