## 【初级】DNS是什么以及其原理，浏览器从URL到展示经历了哪些？

B-能够回答 DNS 概念
A-能够说明从 URL 到渲染全过程
    - 1.根据域名，进行DNS域名解析；
    - 2.拿到解析的IP地址，建立TCP连接；
    - 3.向IP地址，发送HTTP请求；
    - 4.服务器处理请求；
    - 5.返回响应结果；
    - 6.关闭TCP连接；
    - 7.浏览器解析HTML；
    - 8.浏览器布局渲染；
